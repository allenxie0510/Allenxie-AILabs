
# 部署说明 - Allen's AI 资源导航

## 1. 快速发布到 GitHub Pages
1. 将所有文件上传到 GitHub 仓库。
2. 进入仓库 **Settings > Pages**。
3. 选择 `main` 分支并保存。
4. 稍等片刻即可通过提供的 URL 访问。

## 2. 如何实现跨设备同步数据？
由于本项目不依赖服务器数据库，我们使用“代码即数据”的策略：
1. 在浏览器中打开您的网页，使用“添加资源”功能。
2. 完成添加后，点击右上角的 **“同步到 GitHub”** 按钮。
3. 点击 **“复制代码内容”**。
4. 返回 GitHub，编辑仓库中的 `constants.tsx`，将内容替换为粘贴的内容并提交。
5. 网页会自动更新，新资源将在所有设备上永久可见。

## 3. 如何像桌面软件一样使用？
1. 在 Chrome 浏览器打开网页。
2. 点击地址栏末尾的“安装”图标。
3. 该导航站将作为一个独立的窗口运行，不再显示浏览器搜索栏。
